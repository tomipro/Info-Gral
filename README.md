- Python 3

- UNIDAD 3: CONDICIONALES
- UNIDAD 4: CICLOS
- UNIDAD 5: CADENA DE CARACTERES
- UNIDAD 6: COLECCIONES
- UNIDAD 7: ARCHIVOS


CONTENIDO DEL REPOSITORIO:

- Guias de ejercicios resueltas
- Parcialitos con resolucion
- Examenes finales con resolucion
